Welcome to Citycatpg's documentation!
=====================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Citycatpg creates and runs CityCAT models with data and parameters stored in a PostgreSQL database.
RabbitMQ is used to distribute model runs across multiple servers.
